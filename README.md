# IOS_TodoList

### 프로젝트 정보

- 기간 : 24.08.27 - 24.09.05
- 인원 : 1인
- 간단 요약 : iOS 투두리스트 앱 개발해 할 일 생성, 삭제 기능 구현 
- 사용 기술 : Xcode, Swift

### 구체적 설명

Swift와 Xcode를 공부하면서 MVVM 패턴을 활용한 간단한 To-do list 앱을 만들었습니다.
